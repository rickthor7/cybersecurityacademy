Informasi Tentang Malware

Pelajari tentang berbagai jenis malware yang dapat mengancam keamanan sistem Anda. Memahami ancaman adalah langkah pertama dalam melindungi diri Anda.

Virus
Program yang dapat mereplikasi dirinya sendiri dan menyebar ke file atau program lain di komputer.
Contoh: ILOVEYOU, Conficker, Stuxnet
Trojan
Malware yang menyamar sebagai software legitimate untuk menipu pengguna agar menginstalnya.
Contoh: Emotet, Zeus, CryptoLocker
Worm
Malware yang dapat mereplikasi dirinya sendiri dan menyebar melalui jaringan tanpa intervensi pengguna.
Contoh: WannaCry, SQL Slammer, Code Red
Spyware
Malware yang dirancang untuk mengumpulkan informasi tentang pengguna tanpa sepengetahuan mereka.
Contoh: Pegasus, FinSpy, DarkHotel
Ransomware
Malware yang mengenkripsi file pengguna dan meminta tebusan untuk mendekripsinya.
Contoh: WannaCry, NotPetya, Ryuk
Adware
Software yang secara otomatis menampilkan iklan atau mengarahkan pengguna ke situs iklan.
Contoh: Fireball, Appearch, Gator
